# FLASH MAN Frontend

This is the React + Vite frontend for FLASH MAN.

## What It Does

- Chat with the assistant
- Upload and manage documents
- View session history
- Change local UI settings
- Read help and quick-start guidance

## Main Technologies

- React 19
- Vite
- React Router
- Axios
- Browser speech synthesis and speech recognition APIs

## Development

Run the frontend from this folder:

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- The app is branded as FLASH MAN.
- The chat interface includes speech playback, a stop button, and voice input.
- Settings are stored locally in the browser.
